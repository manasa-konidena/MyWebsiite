  1. The template is divided to sections each section has id that refers to its content like (section id = "about").
  2. The container div holds the section structure.
  3. The "title" div is holding a heading title of each section.
  4. The row div (div class = "row") and divs which have classes like this (div class = "col-md-6") are for responsive "Bootstrap Part".
